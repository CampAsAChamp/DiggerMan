<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* **Visual Studio**


### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/CampAsAChamp/DiggerMan.git
    ```
2. Open `DiggerMan.sln` with Visual Studio
3. Upgrade toolset if prompted
4. Start without Debugging
